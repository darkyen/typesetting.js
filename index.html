typesetting.js

A Javascript library for radical web typography!

Typography on the web is becoming more of a focus with newer technologies arising (new devices, CSS3, HTML5). There have been attempts to circumvent a lack of down-to-the-letter control, but all are lacking key options and require large third-party libraries or scripts to run. This was the reasoning for creating vlettering.js, a light-weight, cross-browser, standalone javascript library that exposes web typography like it has never been before!
